# doq

シンプルなタスクキュー管理アプリ。「次にやること」に集中するためのツール。

## 機能

- **タスクの追加**: テキスト入力してタスクを追加
- **キュー形式の管理**: 最初のタスクが「現在のタスク」としてハイライト表示
- **ドラッグ&ドロップ**: タスクの順番を自由に並び替え
- **完了処理**: 完了ボタンでタスクを消化
- **自動保存**: LocalStorageにデータを永続保存（登録日時も記録）

## 使い方

1. `index.html` をブラウザで開く
2. タスクを入力して「追加」または Enter キーで追加
3. 一番上のタスク（青色）に集中して作業
4. 完了したら「完了」ボタンをクリック
5. 必要に応じてドラッグ&ドロップで優先順位を変更

## 技術仕様

- HTML/CSS/JavaScript のみ（依存ライブラリなし）
- LocalStorage によるデータ永続化
- レスポンシブデザイン対応
